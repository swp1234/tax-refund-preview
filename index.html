📌 이 계산기는?

2025년 귀속(2026년 신고) 연말정산 환급액을 미리 계산해볼 수 있는 시뮬레이터입니다.

📝 입력 방법

• 기본 정보: 세전 연봉과 부양가족 수를 입력합니다
• 4대보험: 급여 기준 자동 계산됩니다 (실제와 다르면 수정 가능)
• 소득공제: 신용카드, 체크카드, 주택 관련 지출을 입력합니다
• 세액공제: 의료비, 교육비, 연금저축 등을 입력합니다

🔢 계산 과정

총급여 → 근로소득공제 → 소득공제 → 과세표준 → 산출세액 → 세액공제 → 결정세액 → 원천징수 비교 → 환급/추가납부

⚠️ 주의사항

• 본 계산은 참고용이며 실제 결과와 차이가 있을 수 있습니다
• 정확한 결과는 국세청 연말정산 간소화 서비스를 이용하세요
• 지방소득세(소득세의 10%)는 별도 표시됩니다
• 특수 공제(장애인, 한부모 등)는 포함되지 않습니다
